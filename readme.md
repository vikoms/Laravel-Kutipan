# Laraquotes
Laraquotes adalah sebuah website untuk saling berbagi quote

# Fitur
* CRUD (create,update,delete) Quote
* Like Quote
* Comment Quote
* Like Comment
* Notifikasi 
* Filter Quote (per kategori)

# Dibuat Menggunakan
* Laravel 6.0
* MYSQL Database
* Bootstrap 4.0
